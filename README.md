# Game Portal

A modern, high-performance game portal built with React, Tailwind CSS, and Framer Motion. Featuring a curated collection of 40+ popular unblocked games, category filtering, and an integrated AI study assistant.

## 🚀 Features

- **40+ Popular Games**: A wide variety of games including Action, Sports, Puzzle, IO, and more.
- **Category Filtering**: Easily browse games by genre.
- **Play Random**: Can't decide? Let the portal choose a random game for you.
- **Panic Mode**: Press the `` ` `` (backtick) key to instantly hide your gaming session behind a fake Google Doc.
- **AI Study Assistant**: Integrated Gemini AI in the Education tab to help with homework and learning.
- **Favorites System**: Save your favorite games for quick access.
- **Responsive Design**: Optimized for both desktop and mobile play.

## 🛠️ Tech Stack

- **Frontend**: React 18
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React
- **AI**: Google Gemini API (@google/genai)
- **Build Tool**: Vite

## 📖 How to Use

1. **Browse**: Scroll through the homepage or use the search bar to find games.
2. **Filter**: Use the category buttons to narrow down your choices.
3. **Play**: Click on any game card to open it in a high-performance iframe.
4. **Favorite**: Click the heart icon on any game to add it to your favorites list.
5. **Education**: Switch to the Education tab to use the AI Study Assistant.
6. **Stealth**: If you need to hide your screen quickly, press the `` ` `` key.
